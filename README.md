- View car details and enquiries  
- Responsive and user-friendly interface

### Development Environment
- **Server:** Apache (via XAMPP)
- **Database:** MySQL (phpMyAdmin)
- **Languages:** PHP, HTML, CSS, SQL
- **IDE Used:** VS Code
- **Version Control:** Git & GitHub

### Description
TechCars2 is a responsive web-based platform designed to manage second-hand and luxury car listings. It allows administrators to handle car entries, view customer enquiries, and display specifications such as mileage, engine, model year, and more.
